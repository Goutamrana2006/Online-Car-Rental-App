# 🚗 LuxDrive | Premium Car Rental Website
LuxDrive is a modern and responsive Full Stack Car Rental Web Application developed using React, Tailwind CSS, Node.js, Express.js, and MySQL.
The project delivers a premium luxury car rental experience with secure authentication, real-time booking functionality, and a professional user interface.

# ✨ Features
✅ Modern & Responsive UI Design
✅ Luxury Car Showcase
✅ Secure User Authentication
✅ Login & Registration System
✅ Real-Time Car Booking
✅ MySQL Database Integration
✅ REST API Integration
✅ Dynamic Booking Storage
✅ Professional Navbar & Hero Section

# 🚀 Installation & Setup
## 1️⃣ Clone Repository
```bash
git clone <your_repo_link>
```
## 2️⃣ Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
## 3️⃣ Backend Setup
```bash
cd backend
npm install
npm run dev
```
---
# 🔐 Environment Variables
Create `.env` file inside backend folder:
```env
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=luxdrive
JWT_SECRET=luxdrive_secret_key
```
---
# 🗄️ Database Setup
Create MySQL database:
```sql
CREATE DATABASE luxdrive;
```
Import required tables:
* users
* cars
* bookings
